### Added

- `Separator` — horizontal / vertical divider built on `@radix-ui/react-separator`.
- `Spinner` — soft SVG loading indicator with `sm` / `md` / `lg` sizes, custom label support, and a `bloom-spin` keyframe in the motion tokens.
